# دليل ورك فلو n8n — بوت واتساب بالذكاء الاصطناعي

## ما هو هذا الورك فلو؟

هذا ورك فلو جاهز لـ **n8n** يربط نظام **WhatsApp Manager** بـ **Google Gemini AI**.  
يستقبل جميع أنواع رسائل واتساب ويردّ عليها بشكل ذكي وطبيعي، مع **ذاكرة محادثة** لكل عميل وتسجيل الأخطاء.

---

## الخطوة 1 — استيراد الورك فلو في n8n

1. افتح **n8n** → من القائمة الجانبية اختر **Workflows**
2. اضغط على زر **⊕ New** ثم **Import from file**
3. اختر الملف المُحمَّل `n8n-workflow-whatsapp.json`
4. سيفتح الورك فلو بـ **10 عقد** جاهزة

> **ملاحظة**: رابط خادمك محقون تلقائياً في عقدة الإرسال. لا تحتاج لتعديل الروابط.

---

## الخطوة 2 — إعداد بيانات الاعتماد

يحتاج الورك فلو إلى **اعتمادَين** فقط:

### أ. مفتاح API لخادم واتساب

1. افتح **لوحة التحكم** → من القائمة الجانبية اختر **مفاتيح API**
2. اضغط **إنشاء مفتاح جديد**
3. أدخل اسماً (مثل: `n8n-bot`) واضغط **إنشاء**
4. **انسخ المفتاح الآن** — لن يُعرض مجدداً

ثم في n8n، افتح عقدة **📤 Send Text Reply**:
1. عند حقل **Credential** اضغط **Create new credential**
2. اختر نوع **HTTP Header Auth**
3. أدخل:
   - **Name**: `X-API-Key`
   - **Value**: المفتاح الذي نسخته
4. اضغط **Save**

### ب. مفتاح Google Gemini AI

1. اذهب إلى [Google AI Studio](https://aistudio.google.com/app/apikey) وأنشئ مفتاح API مجاني
2. في n8n، افتح عقدة **Google Gemini Chat Model**
3. أنشئ Credential جديد من نوع **Google PaLM API**
4. أدخل مفتاح Gemini واضغط **Save**

---

## الخطوة 3 — تفعيل الورك فلو ونسخ رابط Webhook

1. اضغط **Active** (زر التبديل في أعلى يمين الشاشة)
2. افتح عقدة **📥 Receive WhatsApp Message**
3. ستجد قسم **Webhook URLs** — انسخ **Production URL**
   - يبدو هكذا: `https://your-n8n.com/webhook/whatsapp-incoming`

---

## الخطوة 4 — ربط الـ Webhook بجلسة واتساب

1. افتح **لوحة التحكم** → **الجلسات** → افتح جلستك
2. في تبويب **الإعدادات** ابحث عن حقل **Webhook URL**
3. الصق رابط الـ Webhook الذي نسخته
4. اضغط **حفظ**

الآن كل رسالة واردة على هذا الرقم ستُرسَل تلقائياً إلى n8n ويردّ عليها الذكاء الاصطناعي.

---

## الخطوة 5 — اختبار الاتصال

أرسل أي رسالة على الرقم المتصل، مثل:

```
مرحبا
```

يجب أن تستلم رداً ذكياً خلال 3-5 ثوانٍ من Gemini AI.

إذا وصل الرد ← الاتصال يعمل بنجاح ✅

## كيف يعمل الورك فلو

### مخطط التدفق

```
📥 Receive WhatsApp Message
        │
        ├──▶ ✅ Respond 200 OK   (فوري — لمنع إعادة الإرسال)
        │
        └──▶ 🔍 Extract Message Fields
                    │
                    ▼
            🤖 AI Agent (Gemini)
            ├── Google Gemini Chat Model
            └── 📝 Conversation Memory (ذاكرة المحادثة)
                    │
                    ▼
            ⚙️ Format AI Output
                    │
                    ▼
            ⏱️ Wait 2 Seconds   (تجنب حظر واتساب)
                    │
                    ▼
            📤 Send Text Reply
                    │
                    └──error──▶ ⚠️ Log Send Error
```

### شرح كل عقدة

| العقدة | الوظيفة |
|--------|---------|
| 📥 Receive WhatsApp Message | يستقبل الرسائل الواردة عبر Webhook |
| ✅ Respond 200 OK | يرد فوراً بـ OK لمنع إعادة إرسال الرسالة |
| 🔍 Extract Message Fields | يستخرج البيانات ويحوّل الوسائط إلى نص مفهوم للذكاء الاصطناعي |
| 🤖 AI Agent | يعالج الرسالة ويولّد رداً ذكياً باستخدام Gemini |
| 📝 Conversation Memory | يحفظ سياق المحادثة لكل مستخدم |
| Google Gemini Chat Model | النموذج اللغوي المستخدم للرد |
| ⚙️ Format AI Output | يُنسّق رد الذكاء الاصطناعي للإرسال |
| ⏱️ Wait 2 Seconds | تأخير 2 ثانية لتجنب حظر واتساب |
| 📤 Send Text Reply | يرسل الرد إلى العميل عبر API |
| ⚠️ Log Send Error | يسجّل أخطاء الإرسال في حالة الفشل |

## أنواع الرسائل المدعومة

يتعامل الورك فلو مع جميع أنواع الرسائل ويُحوّلها إلى سياق مفهوم للذكاء الاصطناعي:

| النوع | نوع واتساب | كيف يُعالَج |
|-------|-----------|-------------|
| نص | `chat` | يُرسَل مباشرة للذكاء الاصطناعي |
| صورة | `image` | يُحوَّل إلى: *"المستخدم أرسل صورة + الوصف"* |
| فيديو | `video` | يُحوَّل إلى: *"المستخدم أرسل فيديو + الوصف"* |
| صوت مباشر | `ptt` | يُحوَّل إلى: *"المستخدم أرسل رسالة صوتية"* |
| ملف صوتي | `audio` | يُحوَّل إلى: *"المستخدم أرسل مقطع صوتي"* |
| مستند | `document` | يُحوَّل إلى: *"المستخدم أرسل ملف: [اسم الملف]"* |
| موقع | `location` | يُحوَّل إلى: *"المستخدم أرسل موقع: [الإحداثيات]"* |
| ملصق | `sticker` | يُحوَّل إلى: *"المستخدم أرسل ملصق"* |
| بطاقة اتصال | `vcard` | يُحوَّل إلى: *"المستخدم أرسل بطاقة اتصال"* |

## تخصيص الذكاء الاصطناعي

### تغيير شخصية البوت

افتح عقدة **🤖 AI Agent** وعدّل حقل **System Message**:

```
أنت مساعد متخصص في [مجال عملك]. ردود يجب أن تكون [أسلوبك].
أضف أي تعليمات أخرى هنا...
```

### تغيير نموذج Gemini

افتح عقدة **Google Gemini Chat Model** وغيّر الموديل:
- `gemini-1.5-flash` — سريع ومجاني (الافتراضي)
- `gemini-1.5-pro` — أكثر دقة وذكاءً
- `gemini-2.0-flash` — الأحدث والأسرع

## payload الرسائل الواردة (للمطورين)

هذا هو الـ payload الذي يصل لـ n8n من خادم واتساب:

```json
{
  "event": "message.received",
  "sessionId": "my-session",
  "data": {
    "type": "chat",
    "from": "966501234567@c.us",
    "to": "966509876543@c.us",
    "body": "مرحبا",
    "timestamp": 1706000000,
    "mediaUrl": null,
    "fileName": null,
    "caption": null,
    "mimetype": null,
    "lat": null,
    "lng": null,
    "loc": null
  }
}
```

| الحقل | الوصف |
|-------|-------|
| `sessionId` | معرّف الجلسة في لوحة التحكم |
| `data.type` | نوع الرسالة: `chat` / `image` / `video` / `audio` / `ptt` / `document` / `sticker` / `location` / `vcard` |
| `data.from` | رقم المُرسِل بصيغة `966501234567@c.us` |
| `data.body` | نص الرسالة (فارغ للوسائط) |
| `data.mediaUrl` | رابط الوسائط (صورة/فيديو/صوت/مستند) |
| `data.fileName` | اسم الملف للمستندات |
| `data.caption` | تعليق الصورة أو الفيديو |
| `data.lat` / `data.lng` | إحداثيات الموقع (للمواقع فقط) |

## ملاحظات مهمة

- **تأخير 2 ثانية**: ضروري قبل الإرسال لتجنب حظر واتساب للرقم.
- **Respond 200 OK**: يجب أن يُرسَل **فوراً** لإيقاف إعادة المحاولة من الخادم.
- **ذاكرة المحادثة**: تُخزَّن في ذاكرة n8n مؤقتاً — تُمسح عند إعادة تشغيل n8n.
- **كل جلسة** لها Webhook مستقل — يمكن ربط أرقام مختلفة بورك فلوهات مختلفة.
- **Gemini API**: متاح مجاناً بحد يومي كافٍ للاختبار والاستخدام المتوسط.
